JS Secret Hunter

A scope-safe JavaScript asset discovery and sensitive-data review extension for Burp Suite Community and Professional.

It analyzes Proxy history and live traffic, follows in-scope JavaScript imports and source maps with strict resource limits, and reports provider credentials, hardcoded secrets, administrative routes, internal services, APIs, GraphQL, WebSocket, and configuration references.

Values are masked by default. The extension does not execute JavaScript, validate credentials against providers, send findings to third parties, or automatically request URLs outside Target Scope.
