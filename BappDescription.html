JS Secret Hunter

A scope-safe JavaScript asset discovery and sensitive-data review extension for Burp Suite Community and Professional.

It analyzes in-scope JavaScript, HTML, JSON, XML, source maps, and other text responses, follows in-scope script imports with strict resource limits, and reports provider credentials, hardcoded secrets, administrative routes, internal services, APIs, GraphQL, WebSocket, and configuration references.

Findings can be scanned from Burp's context menu and reviewed findings can be added to the Site map as redacted audit issues. Values are masked by default. The extension does not execute JavaScript, validate credentials against providers, send findings to third parties, or automatically request URLs outside Target Scope.
