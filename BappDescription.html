JS Secret Hunter

A scope-safe JavaScript asset discovery, sensitive-data review, and client-side vulnerability-candidate extension for Burp Suite Community and Professional.

It locally analyzes all stored Proxy History and live JavaScript, HTML, JSON, XML, source maps, and other text responses, follows in-scope imports, exports, worker scripts, and source maps with strict resource limits, and reports provider credentials, hardcoded secrets, administrative routes, internal services, APIs, GraphQL, WebSocket, general application endpoints, configuration references, and security-sensitive JavaScript patterns.

A Caido-inspired Overview dashboard summarizes triage progress, critical exposure, vulnerable code candidates, sensitive files, attack surface, scan state, and recent signals. Findings, files, links, and assets remain searchable and sortable. Vulnerability checks cover DOM XSS sinks, dynamic code execution, client-side redirects, wildcard postMessage, disabled TLS validation, sensitive Web Storage, prototype-pollution keys, and cleartext WebSockets; all are candidates requiring manual review. Relative links are resolved against their source, sensitive query parameters stay redacted, and endpoints can be prepared in Repeater without being sent automatically. Proxy History notes are optional and disabled by default. The extension does not execute JavaScript, validate credentials against providers, send findings to third parties, or automatically request URLs outside Target Scope.
